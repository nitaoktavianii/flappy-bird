# flappy-bird 🐤🎮

🎯 Features
- Responsive fullscreen canvas (desktop & mobile).
- Control with SPACE or CLICK/TAP.
- 3 Lives ❤️, lose when colliding with pipes or ground.
- Win after scoring 10 points 🏆.
- Simple WebAudio API sound effects (flap, score, hit).
